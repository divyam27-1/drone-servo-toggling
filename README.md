Visualization of Spray Motor Toggling using Point Spray Method

# How to Run

1) Install dependencies with `uv sync`
2) Source the venv with `.venv/Scripts/activate`
2) Run the motor_toggle.ipynb to see a visualization of the motor toggling algorithm working, and generate the motor_toggle_points.json middleware file
3) Run the mission_waypoints.py file to generate the QGC compliant .waypoints and .kml file
